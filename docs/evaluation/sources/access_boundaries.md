# 访问边界备忘

系统的授权边界以用户为单位。所有笔记本、来源、会话和消息都从笔记本的 owner_id 回溯到当前认证用户。接口收到一个 UUID 并不代表请求者拥有该对象；不属于当前用户的对象应像不存在一样返回 404。

浏览器只保存自己的访问令牌，模型密钥只保存在后端环境变量。客户端不能选择检索到的证据、不能提交引用 ID，也不能直接请求模型供应商。来源文本可能包含恶意提示，因此它只能作为不可信证据放入提示词，而不能改变系统规则。

会话中的 assistant 消息和 Citation 均归属于一个 conversation。引用保存的是源分块 ID、顺序和有限长度的摘录，以便即使后续 UI 改版仍能稳定展示。删除笔记本会级联清理它的来源、会话和派生记录。
