# 学习实验记录规范

本评测使用固定、合成且不含个人资料的语料。每轮实验应记录代码提交、嵌入模型、向量维度、Top-K、运行日期和数据库镜像版本。除非实验目标是比较配置，否则不得在同一轮中同时修改分块长度、重叠长度和检索数量。

评测问题分为事实定位、流程理解和边界条件三类。每个问题指定一个期望来源文件和一组答案关键词。检索指标 Recall@5 的定义是：前 5 个返回分块中至少有一个来自期望来源的问题数，除以全部问题数。

引用正确率只在模型生成了回答时计算。若回答中至少一个已验证的引用来自期望来源，记为正确；引用 ID 不在检索候选集中的情况必须被后端丢弃，不能计入正确。关键词命中率仅是自动化的忠实度筛查信号：回答包含指定关键词不代表人工审核已经通过。

响应时间从开始调用检索到拿到检索结果或回答结果为止，以毫秒记录。报告使用平均值和 P95；网络波动、供应商限流和冷启动应在报告备注中说明。最终的回答忠实度由人工逐题阅读答案、引用摘录与期望事实后确认。
