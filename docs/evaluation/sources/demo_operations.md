# 本地演示操作手册

本地数据库容器使用 pgvector/pgvector:pg18，并把 PostgreSQL 映射到主机 5433 端口。启动数据库后，在 backend 目录执行 `POSTGRES_PORT=5433 uv run alembic upgrade head` 应迁移到最新版本。当前嵌入列使用 1024 维向量，配置的嵌入模型必须返回完全相同的维度。

演示前端通过 Vite 提供，默认地址为 http://localhost:5173；后端开发服务默认地址为 http://localhost:8000，交互式 API 文档位于 /docs。演示者先登录、创建笔记本、上传资料并等待来源显示 ready，再创建会话并提出可由资料直接支持的问题。回答区域会先显示文本，再显示来源名、适用页码和固定摘录。

刷新笔记本工作区后，应能再次看到来源与会话历史。若使用另一个账户访问第一个账户的笔记本、来源或会话，服务应返回 404。若嵌入或聊天 provider 未配置，系统不应把密钥交给浏览器，也不应编造没有证据的答案。
